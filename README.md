# ПКВС — Уборка рабочих мест

Интерактивная презентация «до/после» для демонстрации результатов уборки рабочих мест.

## Стек

- React 19 + TypeScript
- Vite
- Tailwind CSS 4
- Framer Motion

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

Результат в папке `dist/`.
